# AutoStonks
Ever wanted to have your background change to Stonks or Not Stonks on a Windows machine automatically? 
No? 
Well too bad I made it anyways!
![Alt text](stonks.png)

Sets the Windows desktop wallpaper based on SPX (S&P 500) daily performance.
    Place this script in Documents\AutoStonks-main\ alongside stonks.png and notstonks.png. The script by default runs at 9:45 AM and 4:15 PM

# Instructions
1. Download the github repo and unzip/drag and place the "AutoStonks-main" Folder in your "Documents" Folder.
2. Open the unzipped folder under "Documents" and run InstallStonks.bat
3. Security pop-up will most likely happen. You will have to allow this run (or don't its your life).
4. Wait
5. ...
6. Profit?
