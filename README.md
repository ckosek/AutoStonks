# AutoStonks
Ever wanted to have your background change to Stonks or Not Stonks on a Windows machine automatically? No? Well too bad!


Sets the Windows desktop wallpaper based on SPX (S&P 500) daily performance.
    Place this script in Documents\AutoStonks\ alongside stonks.png and notstonks.png.

# Instructions
1. Download the github repo and unzip and place the "AutoStonks" Folder in your "Documents" Folder.
2. Open the unzipped folder and run InstallStonks.bat
